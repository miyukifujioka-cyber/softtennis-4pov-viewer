# ソフトテニス 4人視点ビューア

ソフトテニスの4選手それぞれの一人称視点を同時に確認できるインタラクティブツール。

## 機能
- 俯瞰コート（ソフトテニス公式寸法 23.77m × 10.97m）
- 4選手の一人称視点（前衛 / 後衛 / 相手前衛 / 相手後衛）を同時表示
- 選手ドラッグで自由配置／フォーメーション切替（雁行陣 / 並行陣）
- 打球軌道シミュレーション（ショート / ロブ / ストレート）
- 打球範囲を横半径×縦半径の楕円体で可視化
- POV画面ドラッグで上下左右に視点回転
- スマホ縦画面対応／オフライン動作（Three.js 埋め込み）
